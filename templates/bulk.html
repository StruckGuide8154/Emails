Template Docs

Template Variables

Use double curly braces to insert data from your CSV:

Hello {{name}},

You've been selected for {{event}}!

Best,
Your Team

Available Variables

Auto-created from CSV column headers. Example CSV:

name, email, school, year
John, john@asu.edu, ASU, Junior

Gives you:

  • {{name}} → John
  • {{email}} → john@asu.edu
  • {{school}} → ASU
  • {{year}} → Junior

Email Column

Select which CSV column has the recipient email addresses. This is where each email gets sent.

Rich Text vs Raw HTML

Use the toolbar for visual formatting. Click "Advanced: Edit Raw HTML" below the editor to directly edit HTML source. Use "Pull from Editor" to grab current content, edit it, then "Push to Editor" to apply.

Tips

  • Emails send one-by-one for deliverability
  • Preview uses first CSV row for variable fill
  • Test with a small list first
  • Gmail clips emails over 102KB
